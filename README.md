# AI/ML Data Preprocessing Task

## Overview

This task demonstrates the basic data preprocessing steps required before building a machine learning model. The dataset was cleaned and transformed using Python, Pandas, and Scikit-learn.

## Tasks Performed

* Imported and explored the dataset.
* Checked data types and missing values.
* Handled missing values using mean/median imputation.
* Converted categorical features into numerical values using encoding.
* Standardized numerical features.
* Visualized outliers using boxplots and removed them using the IQR method.
* Saved the cleaned dataset as a new CSV file.

## Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Output

The repository contains:

* Dataset used for preprocessing
* Python notebook/script
* Updated (preprocessed) dataset
* Visualizations of outliers

## Learning Outcome

This project provides hands-on experience with essential data preprocessing techniques that improve data quality and prepare datasets for machine learning models.
